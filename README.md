# SmartWater: Machine Learning for Predicting Water Quality

## Abstract
Water contamination is a major issue affecting public health and environmental sustainability. This project focuses on analyzing water quality using data science techniques. The dataset includes various parameters such as pH, turbidity, dissolved solids, and other chemical properties that influence water safety. Data preprocessing techniques were applied to handle missing values and improve data quality. Exploratory Data Analysis (EDA) and visualization methods were used to identify patterns and relationships between variables. Additionally, a basic machine learning model was implemented to evaluate predictive capability. The results provide insights into water quality conditions and highlight the importance of multiple parameters in determining water safety. This project contributes to Sustainable Development Goal 6: Clean Water and Sanitation by supporting better monitoring of water quality.

---

## Problem Statement
Water pollution is a critical problem that affects both human health and ecosystems. Identifying unsafe water conditions early is essential for prevention. This project aims to analyze water quality data and identify patterns that indicate whether water is safe or unsafe based on parameters such as pH, turbidity, and dissolved substances.

---

## Dataset Source
The dataset used in this project is the **Water Potability Dataset** obtained from Kaggle.  
It contains various water quality parameters and a target variable indicating potability (safe or unsafe water).

---

## Methodology / Workflow
1. Data Collection  
2. Data Preprocessing (handling missing values)  
3. Exploratory Data Analysis (EDA)  
4. Data Visualization  
5. (Optional) Model Development using Random Forest

## Results / Findings
- Most water samples are unsafe compared to safe ones  
- pH values are generally around neutral range (6–8)  
- Turbidity alone is not sufficient to determine water safety  
- Correlation between features is weak  
- The Random Forest model achieved approximately 67% accuracy  
- The model performs better in detecting unsafe water samples
